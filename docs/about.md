# 关于 LITTA

## 我们是谁

LITTA 成立于 2021 年，是一家专注于主动健康服务数字化的科技公司。我们致力于通过 AI 技术重构健康服务的交付方式，让优质的健康指导服务变得可配置、可履约、可核验、可规模化。

## 发展数据（2026年目标）

- **供给侧**：30+ 区域代理、10000+ 月活服务者
- **保险融合**：1000万+ 行业采购收入
- **数据资产**：服务过程 100% 数字化，行为数据可核验、可审计

## 我们的边界

**我们不做：** 医疗诊断系统

**我们做：** 让主动健康服务自动发生、过程可追踪、成果可表达、增长可复制
