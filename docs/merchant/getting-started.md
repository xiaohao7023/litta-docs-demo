# 快速开始

欢迎加入 LITTA 商家中心！本指南将帮助你在 5 分钟内完成入驻并开始经营。

## 什么是 LITTA 商家中心

LITTA 商家中心是面向健身教练、运动场馆、康复师等健康服务提供者的数字化经营平台：

- 🏪 **门店数字化** - 线上展示、预约管理、会员运营一站式解决
- 👥 **会员全生命周期** - 从获客到留存，数据驱动精细化运营  
- 💰 **透明收益体系** - 实时业绩查看、自动结算、快速提现
- 🤖 **AI 智能助手** - LITTA POWER 辅助服务策略、履约提醒、内容生成

## 入驻流程

### 步骤 1：注册账号

访问 [LITTA 商家入驻页面](https://merchant.litta.com) 完成注册：

1. 填写手机号并验证
2. 设置登录密码
3. 选择商家类型（个人/企业）
4. 完成实名认证

> **提示**：企业商家需准备营业执照，个人教练需准备身份证和资质证书。

### 步骤 2：创建门店

登录商家后台，点击「创建门店」：

| 信息项 | 说明 | 示例 |
|--------|------|------|
| 门店名称 | 品牌名+门店名 | 乐刻健身·西湖店 |
| 门店类型 | 健身房/工作室/康复中心 | 健身房 |
| 详细地址 | 精确到门牌号 | 杭州市西湖区xx路xx号 |
| 营业时间 | 每日营业时段 | 06:00-23:00 |
| 门店照片 | 门头、内部环境 | 至少3张 |
| 设施标签 | 器械、淋浴、停车等 | 跑步机、力量区、淋浴 |

### 步骤 3：配置服务

根据你的经营类型，配置相应的服务：

**健身房/工作室：**
- 会员卡（月卡/季卡/年卡）
- 私教课程（按节/按周期）
- 团课（瑜伽/单车/搏击等）

**康复中心：**
- 评估服务（体态/运动功能）
- 康复疗程（按次/按周期）
- 线上咨询

**营养师：**
- 饮食方案（减脂/增肌/慢病管理）
- 线上指导
- 周期服务

### 步骤 4：发布上线

完成配置后，点击「提交审核」：

1. 平台将在 1-2 个工作日内完成审核
2. 审核通过后，门店正式上线
3. 开始接收会员预约

## 首次登录后台

审核通过后，登录 [商家后台](https://merchant.litta.com)：

```
首页概览 → 查看今日预约、业绩数据、待办事项
门店管理 → 修改信息、上传照片、调整营业时间
课程服务 → 新增/编辑服务、设置价格、管理库存
会员中心 → 查看会员列表、跟进记录、标签管理
订单预约 → 处理预约、核销、退改申请
数据中心 → 查看经营报表、业绩趋势、会员洞察
```

## 核心概念

### 服务策略

LITTA 采用「服务策略」模式管理商品交付：

- **服务节奏** - 定义服务周期和触达频率
- **组件装配** - 组合课程、指导、测评等模块
- **履约证据** - 记录服务过程，确保交付质量
- **效果表达** - 可视化呈现服务成果

### 周期性服务

健康服务通常需要持续周期才能见效：

- 减脂周期：4-12 周
- 康复疗程：6-12 次
- 习惯养成：21-90 天

LITTA 支持按周期售卖服务，自动提醒履约节点。

### AI 辅助交付

LITTA POWER 系统为商家提供：

- **策略生成** - 根据会员情况推荐服务方案
- **履约提醒** - 关键节点自动提示干预事项
- **内容生成** - 辅助生成训练计划、饮食建议
- **数据洞察** - 分析会员行为，预测流失风险
