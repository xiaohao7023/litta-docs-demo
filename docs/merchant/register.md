# 商家入驻指南

本文档详细介绍 LITTA 商家入驻的完整流程和注意事项。

## 入驻条件

### 个人教练

- 年满 18 周岁
- 持有相关职业资格证书（健身教练证/康复师证/营养师证等）
- 无不良信用记录

### 企业商家

- 合法注册的企业或个体工商户
- 持有有效营业执照
- 具备相关经营资质（如健身场馆需消防验收合格）
- 法定代表人或授权代理人完成认证

## 入驻流程

### 1. 账号注册

访问 [LITTA 商家入驻](https://merchant.litta.com/register)：

1. 输入手机号，获取验证码
2. 设置 8-20 位登录密码（需包含字母和数字）
3. 阅读并同意《商家服务协议》
4. 点击「注册」完成账号创建

### 2. 实名认证

根据商家类型选择认证方式：

**个人认证：**
- 上传身份证正反面照片
- 进行人脸识别验证
- 上传职业资格证书

**企业认证：**
- 上传营业执照
- 上传法定代表人身份证
- 填写企业对公账户信息（用于验证）

> **审核时间**：个人认证 1 小时内，企业认证 1-2 个工作日

### 3. 创建门店

认证通过后，创建你的第一家门店：

**基础信息：**
```
门店名称：建议格式「品牌名·商圈/地标店」
门店类型：健身房 / 工作室 / 康复中心 / 瑜伽馆 / 游泳馆
经营品类：健身 / 康复 / 营养 / 综合
```

**位置信息：**
```
所在城市：精确到区县
详细地址：街道、门牌号
地图定位：在地图上标记准确位置
```

**联系信息：**
```
客服电话：会员可联系的电话
营业时间：每日营业时间段
```

### 4. 门店装修

完善门店展示信息：

**门店照片（至少 5 张）：**
- 门头照片（清晰展示店名）
- 前台/接待区
- 主要训练区域
- 更衣室/淋浴间（如有）
- 特色设施/器械

**门店介绍：**
- 200-500 字门店简介
- 突出特色、优势、服务理念
- 支持图文混排

**设施标签：**
- 器械设施（跑步机、力量区、自由重量等）
- 配套设施（淋浴、储物柜、饮水机等）
- 特色服务（私教、团课、康复等）

### 5. 配置服务

根据经营类型配置服务商品：

**会员卡：**
| 类型 | 建议定价 | 适用场景 |
|------|---------|---------|
| 月卡 | 199-399 元 | 新客体验、短期健身 |
| 季卡 | 499-899 元 | 常规健身用户 |
| 年卡 | 1499-2499 元 | 长期健身用户 |

**私教课程：**
| 类型 | 建议定价 | 包含内容 |
|------|---------|---------|
| 体验课 | 99-199 元/节 | 体测+1节正式课 |
| 正式课 | 300-600 元/节 | 1对1私教服务 |
| 周期课包 | 按课时优惠 | 12/24/36/48 节 |

**团课：**
- 设置课程名称、类型、难度
- 配置排课时间、教练、容量
- 设置单节价格或包含在会员卡中

### 6. 提交审核

完成以上配置后，点击「提交审核」：

- 平台将在 1-2 个工作日内完成审核
- 审核结果通过短信和站内信通知
- 如有问题，可根据反馈修改后重新提交

## 审核标准

### 门店信息审核

✅ **通过标准：**
- 信息真实、完整
- 照片清晰、真实
- 定位准确
- 介绍规范、无违禁词

❌ **常见驳回原因：**
- 使用网图或虚假照片
- 地址不详细或定位偏差过大
- 门店介绍包含联系方式或外链
- 经营类目与实际不符

### 服务商品审核

✅ **通过标准：**
- 定价合理，符合市场行情
- 描述清晰，无夸大宣传
- 服务内容与门店类型匹配

❌ **常见驳回原因：**
- 价格明显偏离市场（过高或过低）
- 使用绝对化用语（"最好"、"第一"等）
- 承诺疗效或保证效果
- 服务描述与实际能力不符

## 入驻后配置

### 绑定收款账户

审核通过后，绑定收款账户：

1. 进入「财务中心」→「收款设置」
2. 选择账户类型（个人/企业）
3. 填写银行卡信息
4. 完成小额打款验证

### 设置员工账号

如有员工需要登录后台：

1. 进入「门店管理」→「员工管理」
2. 点击「添加员工」
3. 填写员工姓名、手机号、角色
4. 员工收到短信后完成注册

**角色权限：**
| 角色 | 权限范围 |
|------|---------|
| 店长 | 全部权限 |
| 教练 | 查看会员、记录服务、查看业绩 |
| 前台 | 处理预约、核销、查看订单 |
| 财务 | 查看数据报表、管理收款账户 |

### 下载商家 App

- iOS：App Store 搜索「LITTA 商家」
- Android：应用市场搜索「LITTA 商家」
- 或使用微信小程序「LITTA 商家助手」

## 常见问题

**Q：一个账号可以创建多个门店吗？**

A：可以。企业商家可创建多个门店，个人教练如有多场地合作也可创建多个。

**Q：入驻需要缴纳费用吗？**

A：入驻免费。LITTA 按实际成交额收取技术服务费，费率根据类目 5%-15% 不等。

**Q：审核被驳回了怎么办？**

A：查看驳回原因，根据提示修改后重新提交。如有疑问可联系商家客服。

**Q：可以修改门店信息吗？**

A：可以。部分信息（如地址、电话）修改后需重新审核，部分信息（如营业时间）可即时生效。
