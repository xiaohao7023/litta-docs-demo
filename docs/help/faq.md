# 常见问题

## 入驻相关

**Q：入驻需要缴纳费用吗？**

A：入驻免费。LITTA 按实际成交额收取技术服务费，费率根据类目 5%-15% 不等。

**Q：审核被驳回了怎么办？**

A：查看驳回原因，根据提示修改后重新提交。如有疑问可联系商家客服。

**Q：一个账号可以创建多个门店吗？**

A：可以。企业商家可创建多个门店，个人教练如有多场地合作也可创建多个。

## 服务相关

**Q：如何设置服务价格？**

A：在「服务管理」中创建服务时设置价格，支持设置原价和优惠价。

**Q：会员如何预约？**

A：会员通过 LITTA App 或小程序查看可约时段，选择时间后完成预约。

## 结算相关

**Q：多久结算一次？**

A：T+1 结算，即订单完成后次日可提现。

**Q：提现多久到账？**

A：工作日提现，一般 1-2 个工作日到账。
