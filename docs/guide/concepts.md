# 核心概念

了解 LITTA 平台的核心理念和关键概念，帮助你更好地使用平台。

## S2B2C 商业模式

LITTA 采用 S2B2C（Supply-chain platform to Business to Consumer）模式：

```
S (平台) → B (商家) → C (会员)
```

- **S - 平台**：提供技术、流量、支付、数据等基础设施
- **B - 商家**：提供服务供给，包括教练、场馆、康复师等
- **C - 会员**：享受健康服务，获得健康改善

## 六层结构

| 层级 | 主体 | 角色说明 |
|------|------|----------|
| S | 平台 | 规则与能力中枢，商业规则定义者 |
| A | 代理商 | 规模化复制，解决「把样板做成可复制」 |
| G | KA（大客户） | 集团化治理节点，统筹规则与资源 |
| B | 门店/团队 | 承接本地经营动作落地 |
| b | 履约交付主体 | 对交付质量负责（教练、康复师等） |
| C | 会员/用户 | 服务发生与结果回流入口 |

## 周期性服务

### 什么是周期性服务

健康服务不同于一次性交易，需要持续周期才能判断效果：

- **健身训练** - 需要 4-12 周才能看到明显效果
- **运动康复** - 需要 6-12 次治疗才能恢复功能
- **饮食管理** - 需要 21-90 天才能养成习惯

### 周期服务的特点

1. **持续交付** - 不是一次完成，需要多次服务
2. **过程记录** - 需要记录每次服务的关键数据
3. **效果评估** - 周期结束时评估整体效果
4. **续费转化** - 基于效果决定是否继续服务

## 服务策略

### 定义

服务策略定义了商品承诺如何交付，包含：

- **服务节奏** - 多久一次、持续多久
- **组件装配** - 包含哪些服务内容
- **触达规则** - 如何提醒和跟进
- **复评机制** - 如何评估和调整

### 示例：减脂服务策略

```yaml
服务周期: 12周
服务频率: 每周3次
组件:
  - 体测评估（开始/中间/结束）
  - 私教训练（每次60分钟）
  - 饮食指导（每日打卡）
  - 进度复盘（每周1次）
触达:
  - 训练前1小时提醒
  - 饮食打卡提醒
  - 周复盘提醒
复评:
  - 第4周中期评估
  - 第8周调整方案
  - 第12周总结效果
```

## 履约证据链

### 什么是证据链

服务发生的关键事件与反馈记录，可追溯、可解释、可核验：

- **预约记录** - 何时预约、什么服务
- **签到记录** - 是否按时到店
- **服务记录** - 做了什么、做了多久
- **反馈记录** - 会员感受、效果评价
- **数据记录** - 体重、体脂、力量等数据

### 证据链的价值

1. **对会员** - 清晰看到服务过程和效果
2. **对商家** - 证明服务价值，减少纠纷
3. **对平台** - 评估服务质量，优化匹配

## LITTA POWER

### AI Native 服务交付系统

LITTA POWER 是商家的幕后 AI 助手：

| 模块 | 能力 | 典型场景 |
|------|------|---------|
| 策略侧 | 辅助生成与优化服务策略 | 新会员进场，推荐服务方案 |
| 履约侧 | 基于证据链做提示、纠偏与建议 | 关键节点提示干预事项 |
| 表达侧 | 辅助生成成果表达与经营素材 | 把阶段成果输出为可传播素材 |
| 运营侧 | 辅助任务化运营与知识支持 | 减少非核心消耗，节奏化经营 |

### 可解释、可复盘、可托底

- **可解释** - AI 的每个建议都有依据
- **可复盘** - 服务过程全程记录，可回顾分析
- **可托底** - 关键决策有人工审核兜底

## 数据闭环

### 数据采集

- **行为数据** - 预约、签到、训练记录
- **生理数据** - 体重、体脂、心率、力量等
- **反馈数据** - 满意度、主观感受、效果评价

### 数据应用

- **会员画像** - 了解会员特征和需求
- **服务优化** - 基于数据调整服务策略
- **风险预警** - 预测流失风险，提前干预
- **效果验证** - 用数据证明服务价值

## 保险融合

### 运动换保障

LITTA 与保险公司合作，实现「运动换保障」：

```
保险为健康服务付费
    ↓
运动数据换保费折扣/权益
    ↓
健康改善降低赔付
    ↓
再投入激励形成正向循环
```

### 对商家的价值

- **新增收入来源** - 保险采购服务
- **降低获客成本** - 保险渠道导流
- **提升服务价值** - 数据可核验，效果可量化
