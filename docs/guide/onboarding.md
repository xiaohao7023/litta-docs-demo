# 商家入驻

完成入驻流程，开启你的数字化经营之旅。

## 入驻条件

### 个人教练

- 年满 18 周岁
- 持有相关职业资格证书
- 无不良信用记录

### 企业商家

- 合法注册的企业或个体工商户
- 持有有效营业执照
- 具备相关经营资质

## 入驻流程

### 1. 账号注册

访问 [LITTA 商家入驻](https://merchant.litta.com/register)：

1. 输入手机号，获取验证码
2. 设置 8-20 位登录密码
3. 阅读并同意《商家服务协议》
4. 点击「注册」完成账号创建

### 2. 实名认证

**个人认证：**
- 上传身份证正反面照片
- 进行人脸识别验证
- 上传职业资格证书

**企业认证：**
- 上传营业执照
- 上传法定代表人身份证
- 填写企业对公账户信息

> **审核时间**：个人认证 1 小时内，企业认证 1-2 个工作日

### 3. 创建门店

认证通过后，创建你的第一家门店：

**基础信息：**
- 门店名称、类型、地址
- 营业时间、联系电话
- 上传门店照片（至少 5 张）

### 4. 提交审核

完成以上配置后，点击「提交审核」：

- 平台将在 1-2 个工作日内完成审核
- 审核结果通过短信和站内信通知
